# Doomsday Bot PC

Приватный Windows-бот для Doomsday: Last Survivors в LDPlayer 9. Программа работает через ADB, ищет игровые элементы по шаблонам и выполняет выбранные рутинные задачи.

## Скачать готовую версию

1. Откройте раздел **Releases** репозитория.
2. Выберите релиз **Последняя portable-сборка**.
3. Скачайте `DoomsdayBotPortable.zip`.
4. Распакуйте ZIP полностью и запустите `DoomsdayBotPortable.exe`.

Не запускайте EXE прямо из архива и не отделяйте его от папки `_internal`.

## Отчёт об ошибке

Нажмите в программе **Создать отчёт**. Полученный ZIP можно передать разработчику: конфиденциальные поля конфигурации автоматически скрываются.

## Разработка

```powershell
py -3.14 -m pip install -r requirements.txt
py -3.14 -m unittest discover -s tests -v
py -3.14 build_portable.py
```

Каждая отправка изменений в ветку `main` запускает тесты и создаёт новый ZIP в GitHub Actions. Тег `latest` всегда содержит последнюю успешную portable-сборку.

Рабочие `config.json`, логи, отчёты, снимки обучения и папки `build/dist` намеренно не хранятся в GitHub.
